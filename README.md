# Customer Churn Prediction

## Overview
This project predicts customer churn using machine learning and deep learning techniques.

## Dataset
The dataset contains 7043 customer records with features such as tenure, charges, contract type, and services.

## Methods Used
- Data Cleaning and Preprocessing
- Exploratory Data Analysis (EDA)
- Logistic Regression
- Random Forest
- Neural Network (MLP)
- SMOTE for class imbalance

## Evaluation Metrics
- Accuracy
- Precision
- Recall
- F1-score

## Tools & Libraries
- Python
- Pandas
- NumPy
- Matplotlib / Seaborn
- Scikit-learn
